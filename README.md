##  Spiele Projekte

### Shrouded Dales – Action-Adventure Prototype

2D-Sidescroller-Action-Adventure mit der Godot 4.0 Spieleengine.

Features:
- Player & Enemy State Machines für Bewegung, Kampf und KI
- Tilemap-basiertes Leveldesign mit mehreren Bereichen und Layern
- Inventory-, Equipment- und Chest-System
- Audio-Controller mit dynamischer Soundscape und Crossfading
- GUI-Menüs und Parallax-Main-Screen

##  Software Projekte

### Resistor Library

Java-Bibliothek zur Darstellung und Umrechnung von Widerstands-Farbcodes.

Features:
- Umrechnung Farbcodes ↔ Ohm-Werte
- Unterstützung für 4-, 5- und 6-Band-Widerstände
- Berechnung von Toleranzen und Temperaturkoeffizienten
- Swing-GUI-Resistor-Calculator
- Unit-Tests und JavaDocs

**Projektbericht:**  
[Projektbericht öffnen](resistor-library/doc/ResistorLib.pdf)

---

### RFID Time Tracking System

IoT-Zeiterfassungssystem mit Microprozessor und RFID zur automatischen Arbeitszeiterfassung.

Features:
- RFID Login mit RC522 Reader
- Flask API + SQLite Datenbank
- Web-UI für Benutzer- und Logverwaltung
- Docker-basierter Server
- Eigene PCB und 3D-gedrucktes Gehäuse

**Projektbericht:**  
[Projektbericht öffnen](zeiterfassungssystem/doc/version_20240311.pdf)
